雜Java_2020-02-25

雜Java_2020-02-25

程式語言執行在作業系統OS，作業系統OS執行在硬體。

Java可跨平台作業系統，因為先在各種OS(kernel)先裝虛擬機，也就是JVM(JAVA Vurtual Machine)。

JAVA 原是SUN，後來賣給ORACLE。

環境安裝

JDK (JAVA Developer Kit)

JRE (JAVA Runtime Environment)

JDK裝完就有JRE了。開發程式需要JDK，只是執行寫好的程式只需要JRE。

版本名詞介紹

Java SE (Standard)

Java ME (Mobile)

Java EE，包含了SE

bin(binary，二進制)

https://www.oracle.com/tw/java/technologies/javase-downloads.html

JDK下載安裝完之後，重要的檔案有:

  • bin/java
  • bin/javac (compiler編譯器)

IDE 整合開發環境

  • inteliej
  • eclipse
  • vscode

Eclipse操作

  • 把package explorer叫回來: window-show view-package explorer

開始Hello World

  • src-new-package
    • 什麼是package? 影響的是使用者端安裝的位置，為避免眾多開發者重複到相同的命名。在手機….慣例上，把domain、url倒過來寫。e.g. 資策會網站原來是iii.org.tw應寫成tw.org.iii，而之後的。
  • 因為是物件導向，要先從Class開始建立。類別(Class)名稱有大小寫區分，慣例，首字大寫。關鍵字在eclipse會高亮。
  • 慣例，程式碼內的public class名稱與java檔名相同，也因此，一個java檔最外層只有一個public class
  • {} 內寫程式
  • public
  • static 沒有物件觀念
  • void JVM要執行的方法
  • main() 內放參數

extends 繼承

Java關鍵字，全都是小寫。

所以像 String[] 不是關鍵字，是類別。

變數給值，就是初始化(initialized)。

最常用的是int，byte常用於和硬體溝通。
